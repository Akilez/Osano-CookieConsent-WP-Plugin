## ipapi

This plugin optionally uses ipapi to detect a visitor country in the browser so the banner can be limited to EU visitors when location services are enabled.

- Service: [ipapi](https://ipapi.co/)
- Usage in this plugin: browser-side country lookup for `law.countryCode` and EU-only banner display

## Plugin Update Checker

This plugin vendors the Plugin Update Checker library by Yahnis Elsts to support GitHub-hosted plugin updates through WordPress admin.

- Project: [plugin-update-checker](https://github.com/YahnisElsts/plugin-update-checker)

- License: MIT
- Usage in this plugin: GitHub release detection, update metadata, and release-asset based plugin downloads
